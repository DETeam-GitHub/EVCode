**EVCode** 是一个用于通过SMTP发送邮箱验证码的Python第三方库，具备删除记录的功能，提供了丰富的自定义选项。

## 功能

- 通过SMTP服务器发送邮箱验证码。
- 支持删除记录的功能。
- 提供了丰富的自定义选项。
